A full Traktor Pro mapping for the Maschine MK2 — now with 3 modifier layers and a 24-page pad surface. Two-deck STEMS mixing (volume, mute, filter) across five on-screen layouts, plus eight Group buttons (A–H) that switch through three layers (Base / Scene / Pattern) for 24 distinct pad pages: stems, instant loops, beatjump, a full 4×4 remix grid, key/pitch play, tone-play, loop-roll, phrase-jump, a silent loop station, the global loop recorder, an FX1 selector, transport for all four decks, and a complete two-voice drum Pattern Player on FX3/FX4.

Lower buttons: play, jump-to-start (both decks), record, deck-focus toggle, beatjump, Scene/Pattern layer keys, FX1 engage (Solo), mute-all-stems, volume-mode (Dial → focused-deck volume), Note Repeat (Cruise/Auto-DJ), load to Deck A / Deck B, and bidirectional browser scroll on the Dial. Boots to the Full Mix mixer screen.

⚠ ALPHA — actively tested on hardware. Core stems, loops, beatjump, remix, key, transport, browse, the Scene-layer loop-roll and tone-play, the Pattern-layer drum routing, and mute-all are confirmed working. Known gap (fix in progress): the Group G FX-select pads do not fire reliably yet — use the Solo button (FX1 engage) and the FX screen pages in the meantime. Some LED/metering outputs still need hardware confirmation (assign a Traktor Out-Port). Please back up your existing Traktor + Controller Editor settings before importing. A full labeled layout guide (control map + all 24 pad maps + 8 screen maps) is in the download.

Install: load the included .ncc in NI Controller Editor and import the .tsi in Traktor → Controller Manager (assign In + Out ports to all three devices: Stems, Performance, FX).

Compatible with: NI Maschine MK2 (full); NI Maschine MK1 from the same .tsi/.ncc (monochrome, no Dial — alpha, untested on hardware).
You must have: Traktor Pro 4 (stems need Traktor Pro 3+; the drum Pattern Player needs Pro 4) · NI Controller Editor

100% original — no third-party mapping data. Built by reverse-engineering the Traktor format and studying community mappings purely as format references (4 Decks Cue/Fx/Stem, Maschine MK2 2.6.1, MF Twister STEMS, Maschine Mikro VU, Maschine ONE, Xone:K2, S4 Instant JogFX, and others). Originally inspired by DJ Reybans' mapping; no Reybans code remains. Full credits + a labeled layout guide are in the download README. A sanitized public TID reference (Traktor command IDs) is also included as TID_catalog_public.xlsx.
