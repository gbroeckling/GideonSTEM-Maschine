Turn your Maschine MK2 into a full Traktor Pro STEMS controller. Two-deck stem mixing (volume, mute, filter) across five on-screen layouts, plus eight performance pad pages on the Group A–H buttons: stems, instant fixed-size loops, beatjump, a full 4×4 remix grid, key/pitch play, loop-roll, a live main-output VU meter, and transport.

Lower buttons: play, jump-to-start (both decks), record, deck-focus toggle, beatjump, load to Deck A / Deck B, and bidirectional browser scrolling on the Dial. Boots to the Full Mix mixer screen.

⚠ ALPHA — actively tested on hardware. Core functions confirmed working; some LED/VU outputs are still being validated (assign a Traktor Out-Port for LEDs/VU). Please back up your existing Traktor and Controller Editor settings before importing. A full layout guide + diagram is included in the download README.

Install: load the included .ncc in NI Controller Editor, and import the .tsi in Traktor → Controller Manager (assign In + Out ports to all three devices).

Compatible with: NI Maschine MK2
You must have: Traktor Pro 4 (stems need Traktor Pro 3+) · NI Controller Editor

100% original — no third-party mapping data. Built by reverse-engineering the Traktor format and studying community mappings purely as format references (4 Decks Cue/Fx/Stem, Maschine MK2 2.6.1, MF Twister STEMS, Maschine Mikro VU, Maschine ONE, and others). Originally inspired by DJ Reybans' mapping; no Reybans code remains. Full credits + a labeled layout guide are in the download README. Thank you to all those authors.
